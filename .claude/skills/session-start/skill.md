# Session Start

Load and execute the session-start checklist:

1. Read `build/build-personas/checklists/oscar/session-start.md`
2. Execute every step in order.
3. Do not skip steps. The gate at the end is the proof that all steps ran.

This skill is invoked at the beginning of every Oscar session, either automatically via the Orchestrator launcher or manually via `/session-start`.
